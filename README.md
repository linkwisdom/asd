asd
===

支持系统开发 ( assist system development )

## Feature

- 支持自定义

## install

> npm install asd -g

## Usage

### 使用系统命令行

```shell
> asd set email liu@liandong.org
> asd module src/module/app/coreword
```

### 使用内建命令行
- 不需要再输入asd前缀

```shell
> asd
> set email liu@liandong.org
> module src/module/app/coreword
```
