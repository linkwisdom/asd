asd
===

辅助系统开发 ( assist system development )

## Feature

- 自动生成模块所需文件
- 支持多级目录级联创建
- 按照fcfe最新文档规范
- 支持创建单个符合模块规范的文档
- 使用方便、快捷
- 支持项目一键式创建
- 支持内建命令扩展
- 支持系统级命令
- 支持自定义上下文
- 支持自定义模板目录
- 支持命令扩展

## install

> npm install asd -g

## Usage

### 使用系统命令行

```shell
> asd set email liu@liandong.org
> asd module src/module/app/coreword
```

### 使用内建命令行
- 不需要再输入asd前缀

```shell
> asd
> set email liu@liandong.org
> module src/module/app/coreword
```
